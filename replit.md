# Portfolio Website

## Overview
A personal portfolio website for Artjom Japins - a developer and creative technologist. This is a static HTML/CSS/JS website with a modern, elegant design featuring animated elements, floating shapes, and a contact form.

## Project Structure
- `index.html` - Main HTML page with portfolio content
- `style.css` - Styling with animations and responsive design
- `script.js` - JavaScript for interactions and animations
- `server.js` - Simple Node.js static file server for development

## Running the Project
The project runs on a Node.js static file server on port 5000.

### Development
Run the "Portfolio Website" workflow which starts `node server.js`

### Deployment
Configured as a static deployment serving files from the root directory.

## Features
- Hero section with animated title
- About section with feature highlights
- Projects grid (dynamically loaded)
- Contact section with email form
- Responsive design with mobile support
- Floating abstract shapes and decorative elements
